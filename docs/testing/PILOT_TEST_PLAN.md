# DIREKT Pilot Test Plan

## Entry criteria

- approved pilot area/categories;
- provider cohort verified;
- legal/privacy review;
- staging/production separation;
- support/field team trained;
- critical tests and restore exercise passed;
- monitoring and stop controls active;
- Play closed-test distribution.

## Cohorts

Representative customers, sole traders, fixed businesses, mobile/hybrid providers, accessibility participants and operations staff.

## Scenarios

Discovery, trust comprehension, no-location fallback, enquiry/response, provider onboarding, correction, expiry simulation, complaint, field visit and payment if enabled.

## Metrics

Task success, trust interpretation, search success, contact/response, verification turnaround/rework, location accuracy reports, complaint severity, crashes, data use, retention and economics.

## Stop criteria

Security/privacy exposure, misleading trust comprehension, uncontrolled fraud, severe operational backlog, critical crash/ANR or unsafe field process.

## Exit

Pilot report, risk update, decision log, prioritized corrections and explicit go/no-go for production expansion.
