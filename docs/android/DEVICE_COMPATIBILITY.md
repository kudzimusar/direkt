# DIREKT Android Device Compatibility

## Provisional support

- min SDK 24, subject to Phase 1 device research;
- target/compile current required stable levels;
- phones primary;
- tablets/adaptive layouts supported;
- common ARM architectures;
- Google Play distribution first.

## Research sample

Capture anonymized:

- Android version;
- RAM/storage class;
- screen size;
- camera/document quality;
- Play Services availability;
- mobile data stability;
- battery restrictions;
- language/font settings.

## Device matrix

At minimum:

- low-resource supported phone;
- median pilot phone;
- current mainstream phone;
- latest Android;
- minimum supported Android;
- small screen;
- large screen/tablet;
- poor network/low storage;
- OEM with aggressive background restrictions.

## Compatibility behaviour

- no crash if map/Play component unavailable;
- clear camera/file fallback;
- WorkManager constraints;
- graceful image memory use;
- adaptive layout;
- no mandatory high-end sensor.

## Release evidence

Record model/OS, build, scenario, result and defect references.
