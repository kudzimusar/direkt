# DIREKT Verification Levels

Levels summarize completed check groups but never replace individual check details.

| Level | Name | Minimum meaning |
|---|---|---|
| V0 | Draft/unverified | Provider-created data only; not publicly represented as checked |
| V1 | Contact confirmed | Approved phone/email contact check |
| V2 | Identity linked | Current representative identity check linked to provider |
| V3 | Professional/business evidence | Required registration/qualification checks current for selected category |
| V4 | Location checked | Required location/premises evidence approved |
| V5 | Field verified | Required in-person visit completed and approved |

## Rules

- Level is calculated, not manually typed.
- Level applies to a provider/category context, not every service universally.
- Individual check status is always accessible.
- Expiry can reduce a level automatically.
- Field verification does not guarantee future presence or workmanship.
- A mobile provider may legitimately reach V4 through a private base/location check without a public storefront.
- Suspended providers do not display an active promotional level.
- Payment plan does not affect level.

## Display

Use plain-language summary, for example:

> “Identity, plumbing qualification and operating location checked. Premises were not visited.”

Never show only “V4” to customers.
